average.cols                   package:maSigPro                   Documentation

Average columns by index

Description:

     'average.cols' uses the index provided by 'prom.vector' to average columns of a matrix               

Usage:

      average.cols (x, prom.vector, r = 0.7) 

Arguments:

     x: a matrix 

     prom.vector: index vector indicating how colums must be averaged
  
     r: minimal correlation value between columns to compute average
    
Details:

    The repvect argument is used to indicate with vertical lines groups of replicated arrays   
     
Value:

    a matrix of averaged columns
